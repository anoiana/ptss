1 Bayes‚Äôs formula

We shall start with Bayes‚Äôs formula with two events \(\theta\) and \(X\): \[ \Bbb{P}(\theta|X) = \frac{\Bbb{P}(X|\theta)\Bbb{P}(\theta)}{\Bbb{P}(X)}; \tag{1.1} \] where \(\theta\) is parameter of interest and \(X\) represents for observed data. To understand Eq. (1.1) intuitively let us deem the following example

Example 1.1 We have 2 coins A and B with the probability of getting a head is \(0.2\) and \(0.7\), respectively. We randomly choose a coin, flip it \(5\) times and save the result. We also know that coin A is selected with probability \(\theta = 0.5\). We are interested in probability of choosing coin A after observing data.

We know that a selected coin is flipped \(5\) with probability \(p\) in which \(p=0.2\) if coin A is selected and \(p=0.7\), otherwise. Thus, \[ X \sim \mathcal{B}in(5,p), \] where \(X\) is the number of head.

Suppose After experiment has been conducted, we have 3 heads and 2 tails. Thus, the probability with coin A selected is \[ \Bbb{P}(X=3|A) = {5 \choose 3}0.2^30.8^2 = 0.0512, \] and \[ \Bbb{P}(X=3|B) = {5 \choose 3}0.7^30.3^2 = 0.3087, \] if coin B was selected.

Applying Eq. (1.1) we calculate \[ \begin{aligned} \Bbb{P}(A|X=3) &= \frac{\Bbb{P}(X=3|A)\Bbb{P}(A)}{\Bbb{P}(X=3|A)\Bbb{P}(A)+ \Bbb{P}(X=3|B)\Bbb{P}(B)} \\ &= \frac{0.0512(0.5)}{0.0512(0.5)+ 0.3087(0.5)}\\ &= 0.1423 \end{aligned} \tag{1.2} \]

Let us write a generic function to calculate Eq. (1.2) and use the function to calculate when the number of observed Heads is \(x = {(0,1,2,3,4,5)}^{\top}\)

f = function(x){
 p = dbinom(x,5,0.2)
 p1 = dbinom(x,5,0.7)
 p/(p+p1)
}

tibble(`X = x` = 0L:5L, `P(A|X=x)` = sapply(0:5,f))%>%
 mutate(across(where(is.double), round, digits = 3))%>%
 draw_table()

Figure 1.1: Probability coin A is selected given x Heads was observed

Intuitively, we know the probability coin A landing on a head is \(0.2\) moderately smaller than \(0.7\) that is the probability coin B lands on a head . Thus, If the number of head is zero, it can be more likely that coin A was chosen, it probability is \(0.993\), and when the number of head increases this probability is smaller. We see that if all 5 realizations come up with head, we have the probability of \(0.002\), which means the possibility of choosing coin A is not feasible.
