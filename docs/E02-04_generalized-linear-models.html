1 Regression Models For Binary Data

Let us denote response variable as \(\{y_i \in \{0,1\}\}_{i=1,2,\dots,n}\) and \(\{\boldsymbol{x}_i \in \mathbb{R}^{m\times 1}\}_{i=1,2,\dots,n}\) be covariates of subject \(i\). Thus, \[ p = \Bbb{P}(y_i = 1) = \mu({\boldsymbol{\beta}}^{\top}\boldsymbol{x}_i), \quad i = 1,2,\dots,n, \tag{1.1} \] where \(\boldsymbol{\beta} \in \mathbb{R}^{m\times 1}\) is coefficients and \(\mu = \eta^{-1}\) is the inverse link function, and hence \(\eta(\Bbb{P}(y_i=1)) = {\boldsymbol{\beta}}^{\top}\boldsymbol{x}_i\). Thus, \((y_i,\boldsymbol{x}_i)\) is a realization of subject \(i\). To illuminate this scenario to statistics framework, we have \[ y_i \sim \mathcal{Bin}(1,\Bbb{P}(y_i=1)), \\ \Rightarrow \Bbb{E}(y_i) = \mu({\boldsymbol{\beta}}^{\top}\boldsymbol{x}_i), \text{ and} \quad \Bbb{V}(y_i) = \mu({\boldsymbol{\beta}}^{\top}\boldsymbol{x}_i)[1-\mu({\boldsymbol{\beta}}^{\top}\boldsymbol{x}_i)] \]

Some options of \(\eta\) comprise

  • \(\eta = logit(p) = \frac{p}{1-p}\), logistic regression.
  • \(\eta = \Phi^{-1}(p)\), probit regression.
  • \(\eta = \log[-\log(p)]\), log-log regression.

In logistic model, an effect size used widely is odd ratio that is defined as follows \[ OR_1 = \frac{\Bbb{P}(y=1|x_i =1)}{\Bbb{P}(y=0,x_i=1)} = \exp(\beta_0 + \beta_1) \] and \[ OR_0 = \frac{\Bbb{P}(y=1|x_i =0)}{\Bbb{P}(y=0,x_i=0)} = \exp(\beta_0) \] and hence \[ \ln(OR_1) - \ln(OR_0) = \beta_1 \] is relative effect between two groups categorized by binary variable \(x_i\), note that \(\boldsymbol{x}_i \in \mathbb{R}^{1\times 1}\). For more information of probit and log-log link, see Demidenko (2013).

1.2 Logistic-Normal Integral

logistic-normal integral plays an important role in logistic model with normally distributed covariate measurement error. The logistic-normal integral is improper \[ I = I(s,\sigma^2) = \frac{1}{\sigma\sqrt{2\pi}}\int_{-\infty}^{\infty}\frac{e^x}{1+e^x}\exp\Big[-\frac{1}{2\sigma^2}(x-s)^2\Big]dx. \tag{1.2} \]

After standardize variable we obtain \[ I = \frac{1}{\sigma\sqrt{2\pi}}\int_{-\infty}^{\infty}\frac{e^{s+\sigma x}}{1+ e^{s+\sigma x}}e^{-\frac{1}{2}x^2}dx. \tag{1.3} \] Eq. (1.3) can be considered as \[ I = \Bbb{E}_{u \sim \mathcal{N}(0,1)}\frac{e^{s+\sigma u}}{1+ e^{s+\sigma u}} \] \(I\) can be approximated by \[ \Bbb{E}_{u \sim \mathcal{N}(0,\sigma^2)}\Phi(s+u) = \Phi\bigg(\frac{s}{\sqrt{1+\sigma^2}}\bigg), \tag{1.4} \] the proof of Eq. (1.4) can be found on page 339, Demidenko (2013).

Another approximation of logistic-normal integral is to approximate the link function first and then apply the back transformation, we call the approximation of the link function is link-approximation. We knew logistic link function is \[ \ln\bigg[\frac{\mu_L}{1-\mu_L}\bigg], \text{ where} \quad \mu_L(s) = \frac{e^s}{1+e^s}, \] We introduce a function of \(\sigma\), \[ \begin{align} R(\sigma) &= \ln\bigg[\frac{\Bbb{E}_{u \sim \mathcal{N}(0,1)}\frac{\exp[s+\sigma u]}{1+\exp[s+\sigma u]}}{\Bbb{E}_{u \sim \mathcal{N}(0,1)}\frac{1}{1+\exp[s+\sigma u]}} \bigg] \\ &= \ln\Bbb{E}_{u \sim \mathcal{N}(0,1)}\frac{\exp[s+\sigma u]}{1+\exp[s+\sigma u]} - \ln\Bbb{E}_{u \sim \mathcal{N}(0,1)}\frac{1}{1+\exp[s+\sigma u]}, \end{align} \tag{1.5} \] and \(R(0) = s\) If taking derivative of Eq. (1.5), we obtain \[ \frac{dR(\sigma)}{d\sigma} = \bigg[\bigg(E_{u \sim \mathcal{N}(0,1)}\frac{\exp[s+\sigma u]}{1+\exp[s+\sigma u]}\bigg)^{-1}+ \bigg(\Bbb{E}_{u \sim \mathcal{N}(0,1)}\frac{1}{1+\exp[s+\sigma u]}\bigg)^{-1}\bigg]\Bbb{E}_{u \sim \mathcal{N}(0,1)}\bigg[\frac{\exp(s+\sigma u)}{(1+\exp[s+\sigma u])^2}u\bigg]. \tag{1.6} \]

Since \(u \sim \mathcal{N}(0,1)\), \(\Bbb{E}(u) = 0\). Hence, \(\frac{d^2R(\sigma)}{d\sigma^2}\bigg|_{\sigma=0} = 0\). We also find second derivative at \(\sigma=0\), \[ \begin{align} \frac{d^2}{d\sigma^2}R(\sigma)\bigg|_{\sigma = 0} &= \bigg[\frac{1+e^s}{e^s}+1+e^s\bigg]\Bbb{E}_u\bigg[\frac{e^s(1-e^2)}{(1+e^2)^3}u^2\bigg] \\ &= \bigg[\frac{1+e^s}{e^s}+1+e^s\bigg]\frac{e^s(1-e^s)}{(1+e^s)^3} \\ &= \frac{1-e^s}{1+e^s}, \end{align} \tag{1.7} \] since \(\Bbb{E}(u^2) = 1\).

proof of Eq. (1.7):

Let \(dR(\sigma)/d\sigma = (k_1^{-1}+k_2^{-1})k_3\) with \(k_1, k_2\) and \(k_3\) are relative to each term in Eq. (1.6). Hence, \(d^2R(\sigma)/d\sigma^2 = (k_1^{-1}+k_2^{-1})'k_3 + (k_1^{-1}+k_2^{-1})k'_3\).

We have \((1/v)' = -\ln(v)'/v\). Let \(v = \Bbb{E}_u\frac{e^{s+\sigma u}}{1+ e^{s+\sigma u}}\), and then \[ \bigg[\bigg(E_{u \sim \mathcal{N}(0,1)}\frac{\exp[s+\sigma u]}{1+\exp[s+\sigma u]}\bigg)^{-1}\bigg]' = \frac{\Bbb{E}_{u \sim \mathcal{N}(0,1)}\bigg[\frac{\exp(s+\sigma u)}{(1+\exp[s+\sigma u])^2}u\bigg]}{\bigg(\Bbb{E}_{u \sim \mathcal{N}(0,1)}\frac{\exp[s+\sigma u]}{1+\exp[s+\sigma u]}\bigg)^2}\bigg|_{\sigma=0} = 0 \] Similarly, if \(v = \Bbb{E}_u\frac{1}{1+ e^{s+\sigma u}}\), we also have \[ \bigg[\bigg(E_{u \sim \mathcal{N}(0,1)}\frac{1}{1+\exp[s+\sigma u]}\bigg)^{-1}\bigg]' = \frac{\Bbb{E}_{u \sim \mathcal{N}(0,1)}\bigg[\frac{\exp(s+\sigma u)}{(1+\exp[s+\sigma u])^2}u\bigg]}{\bigg(\Bbb{E}_{u \sim \mathcal{N}(0,1)}\frac{1}{1+\exp[s+\sigma u]}\bigg)^2}\bigg|_{\sigma=0} = 0. \]

Thus, \((k_1^{-1}+k_2^{-1})'k_3 = 0\), and \(d^2R(\sigma)/d\sigma^2 = (k_1^{-1}+k_2^{-1})k'_3\). We see that \[ k_3' = \Bbb{E}_{u \sim \mathcal{N}(0,1)}\bigg[\frac{e^s(1-e^s)}{(1+e^s)^3}u^2\bigg] = \frac{e^s(1-e^s)}{(1+e^s)^3} \tag{1.8} \] since \(\Bbb{E}(u^2) = 0\). and \[ k_1^{-1}+k_2^{-1}\bigg|_{\sigma=0} = \frac{1+e^s}{e^s}+1+e^s. \tag{1.9} \] From Eq. (1.8) and Eq. (1.9) we obtain Eq. (1.7). \(\quad \blacksquare\)

Also, we have \[ \frac{dR(\sigma)}{d\sigma} = \frac{dR(\sigma)}{d(\sigma^2)}\frac{d(\sigma^2)}{d\sigma} = \frac{dR(\sigma)}{d(\sigma^2)}(2\sigma), \] so \[ \frac{d^2R(\sigma)}{d\sigma^2} = 2(\sigma^2)^{1/2}\frac{d}{d\sigma}\frac{dR(\sigma)}{d(\sigma^2)} + 2\frac{dR(\sigma)}{d(\sigma^2)} \] as \(\sigma^2 =0 \Rightarrow \sigma = 0\), we see the first term on right-hand side equals zero no matter what \(\frac{d}{d\sigma}\frac{dR(\sigma)}{d(\sigma^2)}\) is, and hence \[ \frac{dR(\sigma)}{d(\sigma^2)}\bigg|_{\sigma^2=0} = \frac{\frac{d^2R(\sigma)}{d\sigma^2}\bigg|_{\sigma=0}}{2} = \frac{1}{2}\frac{1-e^s}{1+e^s} \tag{1.10} \]
